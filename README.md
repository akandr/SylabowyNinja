# Sylabowy Ninja

Zagraj online: <https://akandr.cc/sylabowy_ninja/>

Prosta gra dla dzieci uczących się czytać.

## Sterowanie

- Strzałki / WSAD — ruch (na telefonie DPAD)
- Spacja lub dotknięcie planszy — podpowiedź (lektor powtarza sylabę)
- P — pauza
- R — powrót do domku
